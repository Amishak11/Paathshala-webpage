## 📚 Paathshala Coaching Website

A responsive and user-friendly coaching institute website that helps students and parents explore courses, learn about the institute, and easily connect for inquiries and admissions.

---

## 🚀 Features
- 🏠 Home page with institute introduction  
- 👩‍🏫 About section (Founder & Co-founder)  
- 📚 Courses section with program details  
- 🎉 Events and activities section  
- 🏆 Extra-curricular activities  
- 📞 Contact page (Phone, Email, Address)  
- 📱 Fully responsive design  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript  

---

## 🎯 Purpose
To create a professional online presence for a coaching institute and improve communication with students and parents.
